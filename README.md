# Автотесты для API создание наборов
Проект содержит тесты для проверки параметра "name" в запросе создания набора '/api/v1/kits'.
## Как запустить тесты
1. Установить зависимости: pip install pytest requests
2. Запустить тесты: pytest
## Файлы проекта
-'configuration.py' - базовый URL и пути запросов
-'data.py' - тестовые данные
-'sender_stand_request.py' - функции отправки запросов
-'create_kit_name_kit_test.py' - тесты
